# React 19 升级 Spec

## Why
当前项目存在 React 18 和 React 19 类型不兼容问题，导致 shadcn 包有大量 TypeScript 错误。通过统一升级到 React 19，可以彻底解决类型兼容问题。

## What Changes
- 将 @lcw-doc/react 包的 React 类型从 18 升级到 19
- 将根项目的 React 依赖升级到 19
- 修复升级后的所有 TypeScript 类型错误
- 确保 shadcn 包与 react 包完全兼容

## Impact
- 影响文件：packages/react/package.json, packages/shadcn/package.json, 根 package.json
- 需要更新类型定义以适配 React 19 的新类型系统

## ADDED Requirements

### Requirement: React 19 升级
系统必须将所有 React 相关依赖升级到版本 19，并修复所有类型错误。

#### Scenario: 类型检查通过
- **WHEN** 运行 `pnpm run typecheck`
- **THEN** 所有 3 个包（core, react, shadcn）都无 TypeScript 错误

## MODIFIED Requirements

### Requirement: 保持功能不变
**原则**：只升级 React 版本和修复类型错误，不修改任何功能代码逻辑。
