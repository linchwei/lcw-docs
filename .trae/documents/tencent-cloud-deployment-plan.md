# 腾讯云服务器部署文档计划

## 目标

创建 `docs/deployment-tencent.md`，作为腾讯云版本的部署教程。与阿里云文档并列，方便用户按需选择。

## 与阿里云文档的差异点

腾讯云和阿里云在服务器初始化、项目部署、CI/CD 配置等部分完全相同，差异仅在云平台特有操作：

| 差异点 | 阿里云 | 腾讯云 |
|--------|--------|--------|
| 云服务器产品名 | ECS | CVM |
| 轻量服务器产品名 | 轻量应用服务器 | 轻量应用服务器 |
| 防火墙/安全组 | 安全组（ECS控制台） | 安全组（CVM控制台） |
| 域名服务 | 万网/阿里云域名 | DNSPod/腾讯云域名 |
| SSL 证书 | 阿里云 SSL 证书（免费） | 腾讯云 SSL 证书（免费） |
| 容器镜像服务 | 容器镜像服务 ACR | 容器镜像服务 TCR |
| Docker 镜像加速 | `<id>.mirror.aliyuncs.com` | `mirror.ccs.tencentyun.com` |
| 价格参考 | ECS 经济型 e ~¥80/月 | CVM 标准型 S5 ~¥50/月（更便宜） |
| 新用户优惠 | 首年优惠 | 新客专享，2核4G 低至 ¥50/年 |
| SSH 密钥管理 | ECS 控制台绑定密钥对 | CVM 控制台 → SSH 密钥 |
| 对象存储 | OSS | COS |
| 服务器登录 | 默认 root | 默认 root（Ubuntu 可选 lighthouse） |

## 文档结构

与阿里云文档保持相同的 5 大部分 16 章节结构，仅修改云平台特有的内容：

### 需要修改的章节

1. **购买与选型** - 改为腾讯云 CVM 产品名、价格、购买链接、新客优惠
2. **安全组配置** - 改为腾讯云 CVM 控制台操作路径
3. **弹性公网 IP** - 改为腾讯云 EIP 配置说明
4. **SSH 密钥登录** - 密钥文件名改为 `tencent_deploy`，添加腾讯云控制台密钥绑定方式
5. **域名与 HTTPS** - 改为 DNSPod 域名解析、腾讯云免费 SSL 证书
6. **Docker 镜像加速** - 改为腾讯云镜像加速地址
7. **GHCR 镜像拉取失败方案** - 改为腾讯云容器镜像服务 TCR 同步方案

### 无需修改的章节（直接复用）

- 服务器初始化（2.2-2.6）
- 项目部署配置（4-7，除镜像加速外）
- GitHub CI/CD 配置（8-12）
- 部署流程操作指南（13-15）
- 故障排查（16，除云平台特有问题外）
